Introduction

Admin 4B is an open source admin template built on top of Bootstrap 4. The source code can be found on the GitHub Repository.

Quick start

For the template work properly you need to add Bootstrap CSS and JavaScript files. Refer to bootstrap documentation for more information. Also, add the Font Awesome CSS file. Then download the latest release of the template Admin 4B.

CSS

Copy-paste the stylesheet <link> into your <head> after the Bootstrap and Font Awesome stylesheets to load our CSS. 

<!-- link to Bootstrap CSS -->
<!-- link to Font Awesome CSS -->
<link rel="stylesheet" href="admin4b.min.css">

JS

Place the following <scripts> near the end of your pages, right before the closing <body> tag. Bootstrap denpencies and plugins must come first and then our JavaScript plugins. 

<!-- scripts to Bootstrap JS dependencies -->
<!-- scripts to Bootstrap JS -->
<scripts src="admin4b.min.js"><scripts>

Initial setup

An example code of the initial setup can be seen below. 

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
    <!-- link to Bootstrap CSS -->
    <!-- link to Font Awesome CSS -->
    <link rel="stylesheet" href="admin4b.min.css">
    <title>Admin 4B</title>
  </head>
  <body>
    <!-- scripts to Bootstrap JS dependencies -->
    <!-- scripts to Bootstrap JS -->
    <scripts src="admin4b.min.js"><scripts>
  </body>
</html>
JavaScript initialization

All components (including bootstrap) are automatically initialized by the template.
