Style

Some colors and animations are available to the sidebar.

Colors

The sidebar come with two basic colors:

  • default: uses the default element colors.
  • dark: applied using the class .sidebar-dark.
@@include('includes/code-examples/layout/sidebar/sidebar-colors.html')

Animations

The sidebar come with two animations:

  • .sidebar-fade: fade in and fade out effects.
  • .sidebar-slide-left: slide from left to right.
