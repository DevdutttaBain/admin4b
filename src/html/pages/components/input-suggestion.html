Input suggestion

The .input-suggestion is a simple and flexible autocomplete component. For displaying the list of items you may use the .list-group (refer to bootstrap documentations for more details). The basic code of this component can be seen below. 

@@include('includes/code-examples/input-suggestion.html')

Options

Options can be passed via data attributes. Append the option name to data-, as in data-event="".

Name Type Default Description
event string focus The input event that triggers the suggestion.
item string - Selector to indicate what elements should be treated as a list item.
text string - Selector to indicate what element contains the text that will be showed in the input when a item of the list is selected.
empty string - Selector to indicate what element should be displayed when the search has not results.
