Input suggestion

The .input-suggestion is a simple and flexible autocomplete component. For displaying the list of items you use the .list-group and to define the selectable items of the list you use .list-group-item-action. Refer to bootstrap documentation for more details about these classes.

You can use simple texts or customized templates for display the items of the list. Also, the component has two modes of operation: static and async.

Mode Description
static Default mode. Used for static items and has a built-in DOM search for items in the list.
async Used for dynamic items that requires DOM manipulation. This mode disable the built-in DOM search.

Follow to the next topcis for details.
