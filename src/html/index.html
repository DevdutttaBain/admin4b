Introduction

Admin 4B is is an open source, admin dashboard template and control panel theme built on top of Bootstrap 4. The source code can be found on the GitHub Repository.

Quick start

For the template work properly you need to add Bootstrap CSS and JavaScript files. Refer to bootstrap documentation for more information. Also, add the Font Awesome CSS file. Then download the latest release of the template Admin 4B.

CSS

Copy-paste the stylesheet <link> into your <head> after the Bootstrap and Font Awesome stylesheets to load our CSS.
