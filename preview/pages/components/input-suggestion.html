Input suggestion

The .input-suggestion is a simple and flexible autocomplete component. For displaying the list of items you use the .list-group and to define the selectable items of the list you use .list-group-item-action. Refer to bootstrap documentation for more details about these classes.

You can use simple texts or customized templates for display the items of the list. Also, the component has two modes of operation: static and async.

Mode Description
static Default mode. Used for static items and has a built-in DOM search for items in the list.
async Used for dynamic items that requires DOM manipulation. This mode disable the built-in DOM search.

Follow to the next topcis for details.

Simple text

This is the most basic way to create the component. The suggestions are displayed whenever the user start to typing in the input element, that is, when the input event is triggered. See the example below:

Customized template

You do not need to restrict yourself by using only simple texts for items of the list. You can customize like you wish.

To the component work properly you need to define what element contains the text that will be displayed in the input when a item of the list is selected. You can do this by setting the attribute option data-text with the desired element selector. See the example below:

Asynchronous

You can use simple text or customized templates for async items, however the built-in DOM search is disabled.

Searching juices...

No results found 

<div class="form-group">
  <label>What juice do you like?</label>
  <div class="input-suggestion">
    <input id="input-suggestion-async-example" type="text" class="form-control" data-toggle="suggestion" data-async>
    <div class="input-suggestion-list">
      <div class="list-group">
        <p class="list-group-item loader">
          <em class="text-muted">Searching juices...</em>
        </p>
        <p class="list-group-item no-results">
          <em class="text-muted">No results found</em>
        </p>
        <!-- Dynamically generated -->
        <!-- <a href="#" class="list-group-item list-group-item-action" tabindex="-1"></a> -->
      </div>
    </div>
  </div>
</div>
